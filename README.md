# Services

This project makes use of the following technologies to provide realtime API and server-side rendered views.

- Typescript
- NextJS
- HapiJS
- MongoDB
- Redis
- SocketIO
- Swagger / OpenAPI
- Docker / Kubernetes

### Development

``` sh
# start dependancies using docker-compose
docker-compose up -d

# install modules and compile
npm i

# run development mode
npm run dev
```
